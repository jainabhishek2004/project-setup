## Convex + Better Auth + TanStack Start

This example shows how to use Better Auth with Convex and TanStack Start.

This is a standalone copy of the official example. Its monorepo-local dependencies
and build scripts have been replaced with published packages and local commands.

### Running the example

1. Install dependencies

```bash
npm install
```

2. If you haven't run this example before, initialize the database

```bash
npx convex dev --once
```

3. Run the example

```bash
npm run dev
```
